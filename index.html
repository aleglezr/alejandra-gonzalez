Tutorías Conalep

OBJETIVO:

Sistematizar el proceso de tutorías para el plantel Conalep 169 del Estado de Nayarit mediante el desarrollo de una plataforma web y una aplicación móvil para minimizar el tiempo de atención de las solicitudes (reportes)

El sitio web fue hecho para el uso de 2 tipos de usuario, el usuario administrador, es decir, aquella persona encargada de administrar todos los catálogos pertenecientes al sistema de tutorías y el usuario coordinador de tutores, es decir, el encargado de canalizar las solicitudes de atención (reportes) a los diferentes tutores del plantel asi como de agendar citas con tutores y padres de familia en caso de requerirse.

Cabe mencionar que dicha plataforma web,solo fue hecha para visualizarse en un ordenador, no fue elaborada bajo los principios mobile first, dado que el plantel no lo requirió asi.

TECNOLOGÍAS EMPLEADAS:
  • Para maquetar el sitio utilicé HTML5, mientras que para el diseño hice uso de CSS3 & Bootstrap. Además, también hice uso de KendoUI para la creación de tablas.
  • Para el desarrollo utilicé Javascript y jQuery, mientras que con Ajax realizaba las peticiones HTTP.
  • Cabe aclarar, que para el desarrollo de este sistema, unicamente estuve encargada del desarrollo Front-End, para este mismo proyecto habia un programador backend, el cual fue el encargado de crear un servicio web en .NET, entonces el solo me brindaba la documentación de como funcionaba el servicio, y yo solo hacia el consumo de las APIS.

Recepción de solicitudes

Ver más
OBJETIVO:

Sistematizar el proceso de recepción de solicitudes en ventanilla única para el Instituto Nacional de Antropología e Historia (INAH) en el estado de Nayarit.

El sistema fue realizado en WPF (Windows presentation Foundation) un framework de desarrollo de .NET, se eligió utilizar este framework dado que INAH Nayarit no contaba con un Servidor para alojar un sitema web, pero aún así, ellos deseaban una interfaz de usuario que diera el sentimiento de estar utilizando un sistema web.

WPF hereda del desarrollo web la utilización de un lenguaje de meta etiquetas para el desarrollo de la interfaz gráfica UI y los estilos.

TECNOLOGÍAS EMPLEADAS:
  • Para maquetar el sistema utilicé XAML, un lenguaje bastante similar a XML o HTML
  • El desarrollo del modelo lo realicé con Entity framework.
